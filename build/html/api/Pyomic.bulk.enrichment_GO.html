Pyomic.bulk.enrichment_GO

Pyomic.bulk.enrichment_GO(gene_list, go_mode='Bio', organism='Human', description='test_name', outdir='enrichment_go', cutoff=0.5)[source]

Gene enrichment analysis of GO

Parameters:

  • gene_list (list) – The gene set to be enrichment analyzed

  • go_mode (str) – The module of GO include:’Bio’,’Cell’,’Mole’

  • organism (str) – Select from (human, mouse, yeast, fly, fish, worm)

  • description (str) – The title of enrichment

  • outdir (str) – The savedir of enrichment

  • cutoff (float) – Show enriched terms which Adjusted P-value < cutoff.

Returns:

result (pandas.DataFrame) – stores your last query
